Privacy Policy

This application does not save any data used to log into your Spotify account. This application is open-source and can be viewed at it's Github Repository

API Usage

This application makes use of two public APIs.

  • Spotify API
  • Discogs API
The applications uses the Spotify API to get all of a users saved albums, and fetches their community scores from the Discogs API (Unless the score is already saved in the database).
